# Vector Play

**Vector Play** is a **2D game engine for the web** built on **SVG + the DOM**—not a standalone game. It targets browser games that use vector graphics, scene layout, and interactions authored in **TypeScript** and **Svelte**, with **Vite** for development and builds.

The kitchen / egg assets in this repository are **example content** to exercise the engine; replace them with your own scenes and SVG definitions.

## Stack

- Svelte 5  
- Vite  
- TypeScript  

## Development

```bash
npm install
npm run dev
```

```bash
npm run build
npm run preview
```

```bash
npm run check
```
